# 约厨
2018年暑假Android实训项目

## 目录结构
```c
    ┠--  login        // 登录
    ┠--  reigster     // 注册
    ┠--  Page_index   // 首页
    ┠--  page_yuechu  // “约厨”功能模块
    ┠--  data         // 事物抽象对象类包
    ┠--  db           // 数据库操作包
```
