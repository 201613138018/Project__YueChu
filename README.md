# 约厨
2018年暑假Android实训项目

## 功能介绍
    可以用于记录美食的照片/视频，存入数据库并可以进行播放实现，包括菜品的做法的基本实现。

## 目录结构
```c
    ┠--  login        // 登录
    ┠--  reigster     // 注册
    ┠--  Page_index   // 首页
    ┠--  page_yuechu  // “约厨”功能模块
    ┠--  data         // 事物抽象对象类包
    ┠--  db           // 数据库操作包
```
